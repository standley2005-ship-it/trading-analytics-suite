# Signal Logic

## Prototype Indicator Idea

The prototype signal combines:

- Trend direction
- Momentum confirmation
- Risk filter
- Exit conditions

## Example Long Setup

- Price is above the trend baseline.
- Momentum confirms upward movement.
- The trade has a defined stop-loss level before entry.
- The risk-to-reward idea is documented before taking the trade.

## Example Exit Logic

- Price closes below the trend baseline.
- Momentum weakens.
- A pre-defined stop or take-profit level is reached.

## Important Note

Signals should be tested before use. This project is for documentation and learning, not guaranteed trading performance.
